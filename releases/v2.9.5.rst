SIINEOS 2.9.5

SIINEOS
-------

* Improvements / bugfixes:
	- Fixed alert rules with repetition to properly stop when alert condition no longer is met
	- Improved firewall configuration for TOSIBOX Lock-for-Container app to properly work on devices with slower CPUs
	- Improved appearence of file dialogs in web apps
	- Added I/O signal data types UINT8/INT8 (unsigned/signed byte)
	- Added basic support for reading strings in S7 client and connect it to other I/O signals
	- Added support for publishing JSON and string values via MQTT
	- Added support for writing strings to OPC UA value nodes with type STRING
	- A new aggregation mode "Median" has been added to the I/O signal processing page
	- Fixed broken RS485 access for HUB-GM100 and IM18-CCM60
	- Always turn off digital outputs, LEDs, relays and power supplies when I/O unit or corresponding I/O signal is disabled or removed
	- Improved licensing page UI
	- Fixed infinite timeouts when connecting to non-reachable SQL servers
* Components:
	- Bootloader 2021.07.10 (not installed/updated automatically)
	- Linux Kernel 5.4.302
	- Docker 25.0.5 / containerd 1.6.31
	- FreeTDS 1.5.9
	- InCore 2.9.5
	- Qt 5.15.18 with SQLite 3.51.0
	- open62541 1.4.13-1
	- RAUC 1.13.0
	- VictoriaMetrics 1.131.0
	- WebApp Engine 2.9.5
