SIINEOS
-------

* Improvements / bugfixes:
    - Improved timeseries management to better handle metric names with special characters
    - Improved performance and responsivity when deleting timeseries
    - Use ttyUSB0 per default for Modbus RTU (client + I/O endpoint) if bus interface "Serial port" is selected
    - Suppress any communication with IO-Link master if unit is disabled
* Components:
	- Bootloader 2024.04.1 (not installed/updated automatically)
	- Linux Kernel 6.18.22
	- Docker 29.4.0 / containerd 2.2.3
	- FreeTDS 1.5.16
	- InCore 2.10.1
	- Qt 5.15.18 with SQLite 3.51.3
	- open62541 1.4.16-1
	- RAUC 1.15.1
	- VictoriaMetrics 1.140.0
	- WebApp Engine 2.10.1
