SIINEOS Manual
==============

SIINEOS is a secure lightweight Linux-based operating system for embedded IoT/IIoT devices. It's designed to run `InCore applications <https://incore.readthedocs.io>`_ and `Docker containers <https://www.docker.com/>`_.

This manual describes administration procedures and commands for SIINEOS.
